##Processing
DHC data for Illinois was retrieved from the Census with a Python scripts.
The Census Tract data is available by county for all counties in Illinois.  The script extracted the data for all counties in Illinois.
The selected variables were extracted from the Census API. 
Each field represents an estimate for a particular variable.
Processing was primarily completed using the pandas library.

The processing script for the DHC data can be found on [GitHub](https://github.com/nonpartisan-redistricting-datahub/dhc/)
